# ZYN

ZYN is an AI voice assistant built using Python, Arduino, Ollama (Llama 3), speech recognition, and text-to-speech.

## Features
- Voice input
- AI conversations
- Conversational memory
- Arduino button activation
- Spoken responses

## Tech Stack
- Python
- Ollama (Llama 3)
- SpeechRecognition
- gTTS
- PySerial
- Arduino
