# ZYN

ZYN is an AI voice assistant built using Python, Arduino, Ollama (Llama 3), speech recognition, and text-to-speech.

## Features
- Voice input
- AI conversations
- Conversational memory
- Arduino button activation
- Spoken responses

## Tech Stack
- Python
- Ollama (Llama 3)
- SpeechRecognition
- gTTS
- PySerial
- Arduino

## How It Works

1. Press the Arduino button
2. ZYN starts listening
3. User speaks
4. Llama 3 generates a response
5. ZYN replies using text-to-speechg
